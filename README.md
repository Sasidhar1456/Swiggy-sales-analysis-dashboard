## Project Overview

This project focuses on analyzing Swiggy food delivery data using Microsoft Excel to uncover key business insights such as sales performance, customer behavior, and regional trends.

An interactive dashboard was built using advanced Excel features to provide a clear and dynamic view of business metrics, enabling data-driven decision-making.

---

## Tech Stack

- Microsoft Excel  
  - Pivot Tables  
  - Charts & Visualizations  
  - Slicers & Filters  
  - Conditional Formatting  
- Data Cleaning & Transformation  
- Business Intelligence & Reporting  

---

## Dataset

The dataset includes key attributes such as:

- Order Details  
- Sales Amount  
- Ratings  
- Order Quantity  
- Food Categories  
- State-wise Data  

---

## Data Cleaning Process

- Removed duplicate entries  
- Handled missing and inconsistent values  
- Standardized formats (dates, categories)  
- Validated numerical fields (sales, ratings, orders)  
- Created derived metrics for analysis  

---

## Dashboard Features

The Excel dashboard includes:

- **Total Sales KPI**  
- **Average Rating**  
- **Average Order Value**  
- **Total Orders & Rating Count**  
- **Monthly Sales Trend**  
- **Daily Sales Trend**  
- **Sales by Food Type (Veg / Non-Veg)**  
- **State-wise Sales Distribution**  
- **Top Cities by Sales**  
- **Weekly Sales Trend**  
- **Category & Restaurant Filters (Slicers)**  

---

## Key Business Insights

- Identified peak sales periods across months and weeks  
- Analyzed customer preferences (Veg vs Non-Veg)  
- Determined top-performing cities and states  
- Observed consistent daily and weekly order patterns  
- Evaluated customer satisfaction using ratings
